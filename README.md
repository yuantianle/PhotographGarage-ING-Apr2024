# PhotographGarage-ING-Apr2024

## 1. Introduction

This project is using typical web script languages and AWS backend services to build a simple photo album website. The website is designed for users to oauth 2.0 login, view photos. On backend side CRUD operations are supported, with automatically picture resizing and thumbnail generation. Tech stacks include: HTML, CSS, JavaScript, Node.js, Express, AWS S3, AWS Lambda, AWS API Gateway, AWS CloudFront, AWS CloudWatch, AWS CloudFormation.
